# 🚀 ÜSTAT APP - Hızlı Başlangıç

## 📱 3 Adımda Canlıya Al

### 1️⃣ GitHub'a Yükle

```bash
cd flalingo_speaking_app
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/KULLANICI_ADINIZ/ustat-app.git
git push -u origin main
```

### 2️⃣ GitHub Pages Aktif Et

1. GitHub repo → **Settings** → **Pages**
2. Source: **main** branch seçin
3. **Save**

### 3️⃣ Kullan!

⏳ 2-5 dakika bekle

✅ Hazır: `https://KULLANICI_ADINIZ.github.io/ustat-app/`
